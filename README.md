# Fetal Movement Recognition Using Machine Learning and Bayesian Optimization

# Overview

This project focuses on detecting and recognizing fetal movements using sensor data and machine learning techniques. It aims to provide a non-invasive, real-time monitoring system to assist in fetal health assessment.
The system uses accelerometer and gyroscope data to identify movement patterns and classify them using an optimized machine learning model.

# Objectives
* Capture abdominal movement data using sensors
* Preprocess and extract meaningful features
* Classify fetal movements using a Machine Learning model
* Optimize model performance using Bayesian Optimization
* Enable real-time monitoring and visualization

# Technologies Used
* Programming: Python
* Machine Learning: MLP (Multi-Layer Perceptron)
* Optimization: Bayesian Optimization
* Sensors: ADXL Accelerometer, Gyroscope
* Platform: ThingSpeak (for real-time data visualization)

# System Architecture
1. Sensor data collection (accelerometer + gyroscope)
2. Data preprocessing (noise filtering, normalization)
3. Feature extraction
4. Model training using MLP classifier
5. Hyperparameter tuning using Bayesian Optimization
6. Real-time data upload to ThingSpeak

# Methodology
* Sensor data is collected from simulated abdominal movements
* Data is cleaned and processed for consistency
* Features are extracted from raw signals
* MLP classifier is trained on labeled data
* Bayesian Optimization improves model parameters
* Output is visualized for monitoring purposes
  
# Results
* Improved classification accuracy using Bayesian Optimization
* Efficient detection of movement patterns
* Real-time monitoring capability achieved

# Future Improvements

# Applications

* Early detection of fetal distress
